## 구현 기능

<li>lineTo(), stroke()를 이용해 paint 기능 구현</li> 

<li>fillRect()를 이용해 fill 기능 구현</li> 

<li>stroke()를 이용해 line 기능 구현</li> 

<li>fillRect()를 이용해 reset 기능 구현</li> 

<li>createObjectURL()를 이용해 파일 추가 기능 구현 </li>
